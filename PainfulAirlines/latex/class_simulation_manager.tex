Function\+: \mbox{\hyperlink{class_simulation_manager_a5ff3bcee43f2da414a6fa9df17b7572f}{update\+Help\+Status(\+Customer\+Type)}}  For the given customer type, this function retrieves the customer service rep and, if they are busy helping a customer, determines how long they have been helping the customer. If it is within the range of time it should take to help, this function essentially flips a coin to determine whether they will finish this customer at the current time tick. If it is determined its time to finish via coin flip or because the max time it should take to help has been reached, the rep finishes the customer they currently are helping, which sets the amount of time it took for the customer to be helped. The customer is added to the list of customers that have been helped.
